# 🔏 极简文本加密助手 (MinimalTextEncryptor)

一个基于 Python Tkinter 和 AES-256 加密标准的 Windows 桌面工具。专为追求极致隐私和便捷办公的用户打造。

## ✨ 核心特性

- **金融级安全**：采用 AES-256 (CBC 模式) 对文本进行加解密，配合 PBKDF2HMAC 密钥派生算法（10万次迭代）。
- **沉浸式 UI**：基于 Azure Dark 主题，自定义暗黑模式标题栏，完美适配 Windows 任务栏。
- **无缝编辑**：支持 `.wt` 加密文件直接打开，解密至内存并在记事本预览；**关闭记事本即自动检测改动并重新加密**。
- **系统集成**：一键注册 Windows 右键菜单及文件关联（`.wt` 后缀），支持图标显示。
- **极致便携**：单文件 `.exe` 运行，无需安装，解密过程不在磁盘留下明文痕迹。

## 🛠️ 技术栈

- **语言**：Python 3.10+
- **界面**：Tkinter + Azure-ttk-theme
- **加密**：`cryptography` 库
- **系统适配**：`ctypes`, `winreg`

## 🚀 部署指南

### 1. 环境准备
确保已安装 Python 10 或更高版本，并安装依赖：
```bash
pip install cryptography azure-ttk-theme
```

### 2. 编译打包
本项目使用 PyInstaller 打包。使用提供的 `.spec` 文件以包含图标和主题资源：
```bash
pyinstaller "文本加密助手.spec" --noconfirm
```
编译后的文件位于 `dist/文本加密助手.exe`。

## 📖 功能说明

### 加密流程
1. 启动程序或右键点击 `.txt` 选择打开方式。
2. 输入访问密码。
3. 程序生成 16 字节随机盐值和 IV，将文件加密为 `.wt` 后缀并删除原明文。

### 查看与修改
1. 双击 `.wt` 文件（首次运行需点击“关联文件”）。
2. 输入正确密码。
3. 内容将临时解密并调起 Windows 记事本。
4. **保存说明**：在记事本中修改内容并保存后，关闭记事本，程序将自动重新加密回原文件。

### 注意事项
- **关于密码**：本程序不存储密码，若密码遗失，加密内容将无法找回。
- **管理员权限**：关联文件后缀功能涉及修改注册表，需以管理员权限运行。
